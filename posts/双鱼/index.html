激活虚拟环境#

打开 Anaconda Prompt

conda activate trans

cd trans

Whisper#

whisper 0.mp4 --word_timestamps True -o 0 --initial_prompt ""
  • 默认 Turbo 模型， --model MODEL 指定模型
  • 默认 srt、txt、json 等全格式输出
  • 默认 CUDA
  • --word_timestamps True 开启字级时间戳
  • --output_dir OUTPUT_DIR, -o OUTPUT_DIR 指定输出文件保存的目录。
  • --initial_prompt INITIAL_PROMPT 指定第一段的初始提示文本，帮助模型生成更符合语境的结果。

流程#

有官方字幕#

  1. 原文提取 选择官方字幕提取文本

  2. 字幕分段 选择转录字幕分段

  3. 字幕提取 输出英文字幕分段文本

  4. 转录纠错 将转录字幕分段比对纠错

  5. 英文字幕 生成自行断句英文字幕 0E.srt

  6. 原文提取 选择 0E.srt 提取须翻译文本 0E.txt，清空 0E 文件夹

  7. 字幕分段 选择 0E.srt 分段

  8. 字幕提取 输出英文字幕文本

  9. 文本翻译 译文提取 译文分句 字幕对齐 中文字幕
